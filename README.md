# RWA Tokenization App

## Overview
Tokenize real-world assets using:
- Thirdweb (contracts + frontend)
- Chainlink (real-world data)

## Setup

### 1. Install
npm install

### 2. Run frontend
cd apps/web
npm run dev

### 3. Deploy contract
Use Thirdweb dashboard or Foundry

### 4. Configure env
Copy .env.example → .env

## Features
- Mint property NFTs
- Store metadata on IPFS
- Update valuation via Chainlink
